Editor

Aqui será o editor do SQL, com caixa de texto, botões para executar, limpar, terá uma tabela embaixo (como nos sgbds) que terá o retorno das consultas, terá o log do script (para informar erros, sucesso, atenção, etc).

Missões

Aqui se encontrará as missões tipo em check-list que deverão ser feitas pelo jogador.

Tabelas

Aqui será o dicionário do sistema, para saber quais campos existem em todas as tabelas, uniões, tipos de dados, etc., para servir de consulta a qualquer momento.

Ajuda

Uma pequena alternativa para os comandos. Servirá como apoio apenas em comandos básicos, explicando e exemplificando para o jogador.

Sobre

Poderá ter um resumo do projeto, objetivo, perfil dos desenvolvedores.
